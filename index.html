飞鸟阅读

专业的小说内容管理系统，基于 ThinkPHP6 + MySQL + Layui 构建

🚀 快速开始

🚀 快速部署

  • 运行 git clone https://gitee.com/paheng/feiniao.git 克隆项目到本地

  • 访问 http://你的域名/install/index 开始安装向导

  • 15分钟内即可搭建完成一个功能完整的小说阅读网站
